## How to run
* You need Android Studio (at least 2022.1.1) with Kotlin plugin (at least 1.7.21)
* Enter the command in the terminal
```git
git copy https://github.com/askdh-b/Shop
```
* app -> Run or
```gradle
gradlew build
```
